#### 네트워크

- 단말들이 서로 통신 가능하도록 연결해주는 것
- 서로 통신가능한 단말간의 연결

#### 인터넷

- Inter(~간의) + net(네트워크)
- 네트워크 간의 연결을 의미.
- 즉 여러 네트워크가 연결된 것을 인터넷이라 함.

#### 네트워크의 형태

- WAN : LAN과 LAN을 연결한 장거라 통신망 (ISP로 연결, 물리적 네트워크)
- LAN : 근거리 네트워크 (물리적 네트워크)
- VPN : 논리적인 사설망 (A회사와 B회사가 다른 LAN에 있을때 통신이 불가능하다. 통신을 위해서는 라우팅이 필수적이게 된다. 암호화 통신을 기반으로 논리적인 LAN을 구성할 수 있다.)

#### 네트워크 표준기구

- 네트워크 표준기구 : ISO,IEEE,ITU-T
- 인터넷 표준 기구 : IETF, RFC

> 이더넷 표준 : IEEE 802.3, TCP/IP : RFC 1122 & 1123, HTTP/1.1 : RFC 2616

#### 네트워크 구조

- 네트워크 구조는 크게 규모, 서비스 통신방식, 구현 방식등을 통해 설계된다.
> - 규모 : 단말기의 수, 집단의 크기
> - 통신 방식 : Server & Client , P2P
> - 토폴로지 : Star, Ring, Mesh, Bus, Tree, Redundancy
> - 트리 토폴리지의 경우 계층적으로 설계되는 구조 : 라우터에 스위치가 연결되고 스위치에 여러 단말이 연결되는 형태 (트리 구조가 여러개 연결된 형태가 가장 대중적인 구조)
> - 리던던시 토폴로지는 가용성과 연관되어 있음 : 하나의 스위치가 장애가 발생했을때, 이중화된 다른 스위치 장비를 통해 통신하도록 설계한 구조. (대부분의 기업은 해당 구조를 사용함)

> L4 로드밸런서 영역을 DMZ라고 한다. DMZ는 외부와 통신이 가능한 영역이기에 서버가 위치한다. L4 로드밸런서는 트래픽을 여러서버로 균일하게 분산하기 위한 장비이다.

- 클라우드 네트워크 구조 (AWS)

![image](https://user-images.githubusercontent.com/49670068/138584139-ff484725-9c23-45b9-8a65-9aff51af6a7d.png)

> - VPC : 가상 사설 클라우드로써, 논리적인 사설망이다.
> - ELB : 로드 밸런서
> - SG : 보안그룹, 즉 방화벽
> - IGW : 인터넷 게이트웨이 (인터넷과 통신하기위한 장비, VPC는 인터넷과 분리되어 있으므로 IGW를 통해 인터넷과 통신해야한다.)
> - ROUTE53 : DNS (도메인에 대응하는 IP주소를 제공)
